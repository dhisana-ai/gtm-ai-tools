Help & Documentation

This page provides a quick overview of how to use the Dhisana AI Agent web app.

Getting Started

  1. Open the Settings page and fill in your API keys.
  2. Visit Setup & Run instructions for installing dependencies and launching the app.

Running Utilities

Use the Run a Utility page to execute one of the available tools. Select a utility, provide parameters and optional CSV input, then click Run.

Pushing Leads to Dhisana

After running a tool you can send any LinkedIn profile URLs in the output directly to Dhisana for research and outreach.

  1. Create a Smart List in Dhisana and choose the Webhook option.
  2. Copy the webhook URL and generate an API key.
  3. Set DHISANA_WEBHOOK_URL and DHISANA_API_KEY on the Settings page.

A Push to Dhisana AI Contacts button will appear after a run. Click it and Dhisana will score and route the leads automatically.

Vibe Coding New Workflows

You can create additional utilities with the OpenAI Codex CLI.

  1. Install Node.js 22+ and @openai/codex.
  2. Run task add_utility -- my_tool "describe what it should do".
  3. Commit your script and push it to GitHub.

Example ideas:

  • Find people by job title at a company.
  • Look up people by name and keywords.
  • Scrape a custom website for leads.
  • Use an LLM to research a company.

See the vibe coding guide for details.
